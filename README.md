MNE-EXTRAS
==========

This repo is a collection of gists with utils to use with [mne-python](https://github.com/mne-tools/mne-python).

Usage
-----
```
git clone --recurse-submodules $THIS_REPO mne-extras
cd mne-extras
pip install -e .
```
